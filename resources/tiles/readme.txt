Generate Tile Index tile cache.
===============================
Elvis.qgs - It's a project used to generate download indexes for the data coverage of elevation data. It is expected that you have QGIS installed and it is compatible with QTiles. Qtiles requires an older version of QGIS, 2.18.x

  QGIS - https://qgis.org/en/site/forusers/download.html
  QTiles - https://www.qgistutorials.com/en/docs/creating_basemaps_with_qtiles.html

Save the project files somewhere where QGIS expects to read it.
