Purpose

This Geoscience Australia FME Web Service is provided to give spatial data software users and developers a system to check their datum transformations with a standard. It can be used to transform data that references the Geocentric Datum of Australia 1994 (GDA94) to data referenced on the Geocentric Datum of Australia 2020 (GDA2020).

Targets

The 7-parameter similarity transformation is currently available for GDA94 to GDA2020. The NTv2 transformation grid or 7-parameter similarity transformation is available for AGD66/AGD84 to GDA94.
